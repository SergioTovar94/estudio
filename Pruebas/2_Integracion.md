# Pruebas de Integración

Verifican que varios componentes funcionen juntos.

Por ejemplo:

- Controller
- Service
- Repository
- Base de datos

Se prueba:

- Si los datos se guardan
- Si la conexión funciona
- Si el flujo completo responde

| Lenguaje/Framework | Herramienta de pruebas                      |
| ------------------ | ------------------------------------------- |
| Java / Spring      | Spring Boot Test / Testcontainers / H2      |
| Python / Django    | pytest / Django TestCase / SQLite in-memory |
| JavaScript / React | Jest / Supertest / Testing Library          |
