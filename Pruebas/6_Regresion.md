# Pruebas de Regresión

Se hacen cuando:

- Se cambia código
- Se corrige un bug
- Se agrega funcionalidad

| Lenguaje/Framework | Herramienta de pruebas                    |
| ------------------ | ----------------------------------------- |
| Java / Spring      | JUnit / Spring Boot Test / Jenkins        |
| Python / Django    | pytest / Django TestCase / GitHub Actions |
| JavaScript / React | Jest / Cypress / GitHub Actions           |
