# Pruebas de Sistema

Verifica si el sistema completo está funcionando.

Ejemplo:

- API funcionando
- Servicios conectados
- Base de datos activa
- Seguridad aplicada

| Lenguaje/Framework | Herramienta de pruebas                   |
| ------------------ | ---------------------------------------- |
| Java / Spring      | Selenium / REST Assured / Testcontainers |
| Python / Django    | Selenium / pytest / Docker               |
| JavaScript / React | Cypress / Playwright / Selenium          |
