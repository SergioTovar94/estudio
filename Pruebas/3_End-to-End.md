# Pruebas End-to-End (E2E)

Simulan lo que hace un usuario real

Usuario:

- Entra al sitio
- Hace login
- Compra un producto
- Recibe confirmación

Esto prueba:

- Frontend
- Backend
- Base de datos
- API
- Autenticación

Herramientas:

- Selenium
- Cypress
- Playwright

| Lenguaje/Framework | Herramienta de pruebas          |
| ------------------ | ------------------------------- |
| Java / Spring      | Selenium / Playwright / Cypress |
| Python / Django    | Selenium / Playwright / Cypress |
| JavaScript / React | Cypress / Playwright / Selenium |
