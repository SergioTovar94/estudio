# Pruebas de Seguridad

**Verifican:**

- Autenticación
- Autorización
- Vulnerabilidades
- Ataques

Ejemplos:

- SQL Injection
- XSS
- CSRF
- JWT manipulation
- Rate limiting

Herramientas:

- OWASP ZAP
- Burp Suite
- Snyk

| Lenguaje/Framework | Herramienta de pruebas        |
| ------------------ | ----------------------------- |
| Java / Spring      | OWASP ZAP / Burp Suite / Snyk |
| Python / Django    | OWASP ZAP / Bandit / Snyk     |
| JavaScript / React | OWASP ZAP / Snyk / npm audit  |
