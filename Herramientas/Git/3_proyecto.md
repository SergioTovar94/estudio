# 1. Crear el repositorio en Git

# 2. Inicializar git

```bash
git init
```

Eso crea la carpeta .git

# 3. Crear el archivo git ignore

```bash
.gitignore
```

Cada tipo de proyecto tiene sus archivos a excluir

# 4. Agregar los archivos al repositorio

```bash
git add .
```

# 5. Crear el primer commit

```bash
git commit -m "Mensaje"
```

# 6. Conectar con github

```bash
git remote add origin https://github.com/usuario/polizas-api.git
```

# 7. Enviar el repositorio a github

```bash
git branch -M main
```

```bash
git push -u origin main
```
