# Fechas

| Tipo SQL                   | Descripción                                        | Ejemplo                     |
| -------------------------- | -------------------------------------------------- | --------------------------- |
| `DATE`                     | Solo fecha (año-mes-día)                           | `2026-03-16`                |
| `TIME`                     | Solo hora (hora:minuto:segundo)                    | `17:45:30`                  |
| `DATETIME` / `TIMESTAMP`   | Fecha y hora completas                             | `2026-03-16 17:45:30`       |
| `TIMESTAMP WITH TIME ZONE` | Fecha y hora con zona horaria (PostgreSQL, Oracle) | `2026-03-16 17:45:30+05:00` |
