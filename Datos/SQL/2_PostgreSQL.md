# Instalación

## PostgreSQL Server

Es el motor de la base de datos. Guarda los datos, ejecuta consulas, maneja transacciones.

## PgAdmin4

Es la interfaz gráfica (GUI)

¿Qué permite hacer?

- Ver tablas
- Ejecutar queries
- Crear índices
- Administrar usuarios
- Ver estadísticas

## Stack Builder

Un instalador de extensiones y herramientas adicionales

¿Qué instala?

- PostGIS (geoespacial)
- Drivers (ODBC, JDBC)
- Extensiones

## Command Line Tools

Herramientas de consola

Incluye

- psql - Cliente para ejecutar SQL
- pg_dump - Backups
- pg_restore - pg_restore
