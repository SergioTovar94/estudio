# Terminal integrada

`Ctrl + ñ`

# Settings

Hay dos tipos

## Global

Afecta todos los proyectos. Vive en Archivo → Preferences → Settings (User)

## Workspace

Afecta solo esa carpeta (archivo .vscode/settings.json).

# Navegación

Ir a archivo:

```
Ctrl + P
```

Ir a función dentro del archivo:

```
Ctrl + Shift + O
```

Buscar en todo el proyecto:

```
Ctrl + Shift + F
```

✔ Multicursor (muy poderoso)

```
Alt + Click
```

O:

```
Ctrl + D
```

Eso te ahorra muchísimo tiempo editando.

✔ Paleta de comandos (tu mejor amiga)

```
Ctrl + Shift + P
```
