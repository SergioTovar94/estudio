# Funciones

```JavaScript

// Tradicional
function sumar(a, b) {
  return a + b;
}
```

## Funciones flecha - arrow functions

```JavaScript
// Flecha (si es una línea, puedes omitir `return` y llaves)
const sumar = (a, b) => a + b;

// Con bloque de código
const sumar = (a, b) => {
  const resultado = a + b;
  return resultado;
};

// Sin parámetros
const saludar = () => console.log("Hola");

// Con un solo parámetro (paréntesis opcional)
const cuadrado = x => x * x;
```
