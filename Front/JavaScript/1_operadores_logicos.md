# Operadores lógicos

| Nombre                | Sintaxis | Para qué sirve                                                                      |
| --------------------- | -------- | ----------------------------------------------------------------------------------- |
| AND                   | `&&`     | Se usa cuando todas las condiciones deben cumplirse.                                |
| OR                    | `\|\|`   | Se usa cuando basta con que una condición sea verdadera.                            |
| NOT                   | `!`      | Invierte el valor booleano (true → false, false → true).                            |
| Nullish Coalescing    | `??`     | Devuelve el valor de la derecha solo si el de la izquierda es `null` o `undefined`. |
| AND lógico asignación | `&&=`    | Asigna el valor solo si la variable actual es truthy.                               |
| OR lógico asignación  | `\|\|=`  | Asigna el valor solo si la variable actual es falsy.                                |
| Nullish asignación    | `??=`    | Asigna el valor solo si la variable es `null` o `undefined`.                        |
