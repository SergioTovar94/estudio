# Autenticación

¿Quién eres?: La autenticación es el proceso mediante el cual un sistema verifica la identidad de un usuario, servicio o dispositivo.

## Factores de autenticación

### Algo que sabes

- Contraseña
- PIN
- Respuesta secreta

### Algo que tienes

- Token
- Celular
- Tarjeta
- Código OTP

### Algo que eres

- Huella
- Rostro
- Iris
- Voz

## Tipos de autenticación

### Autenticación simple

Un solo factor. Ejempli: usuario + contraseña

### Autenticación Multifactor (MFA)

- Contraseña
- Códifo enviado al célular

### Autenticación contínua

El sistema valida constantemente

- Detección de comportamiento
- Verificación de inicio de sesión

# Autorización

¿Qué puedes hacer?: La autorización es el proceso mediante el cual un sistema determina qué acciones puede realizar un usuario autenticado.

# Principio del menor privilegio

El usuario debe tener solo los permisos necesarios para realizar su trabajo.

# Identidad

Representación del usuario en el sistema

# Credenciales

Datos utilizados para autenticar el usuario

- Contraseña
- Token
- Certificado
- Clave API
- Biometría

# Sesión

Estado que mantiene el sistema después de autenticar el usuario.

# Token

Credencial digital que representa la identidad digital de un usuario después de autenticarse.

# Roles y permisos

Permiso: Acción específica que puede realizar un usuario.
Rol: Conunto de permisos
