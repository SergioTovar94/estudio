# Arquitectura Monolítica

Es la artquitectura más común y más conocida. Allí la aplicación está contenida en un solo modelo de clases, una sola lógica de negocio y una sola base de datos. Todo el código fuente se compila en un solo archivo ejecutable.

Todo el sistema está en una sola aplicación.

- Un solo backend
- Un solo deploy
- Todo el código junto

Ventajas:

- Más simple al inicio
- Fácil de desarrollar

Desventajas:

- Difícil escalar cuando crece
